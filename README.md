# Blockchain-Voting-System

## Overview💜

The Block Chain Based Voting system is a project brought by Coinlocator, In this we used solidty to form a smart contract

## Running VotingDapp Locally👨‍💻

Clone Repo:

```sh
git clone https://github.com/AwesomeTeam1206/Token_voting_website
cd Token_voting_website
```

```Node version: 
Any stable version of Node.js later than v16 is available.
```
Install Dependencies:

```sh
npm install
```

Run The Application:

```sh
npm start
```

Open `localhost:3000` to access the application!😎
